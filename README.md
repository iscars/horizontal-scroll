## Описание
Адаптивное горизонтальное меню. На touch устройствах позволяет осуществлять прокрутку с помощью жестов. 
Ширина элементов списка может быть произвольной.

###
Для сборки проекта использован webpack v3.

##### Инициация:
- `$ npm i`

##### Разработка:
- `$ npm start`

##### Сборка:
- `$ npm run build`
